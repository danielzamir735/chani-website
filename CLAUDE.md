# Project Guidelines

## Tech Stack
- **Framework:** Next.js App Router
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion

## UI/UX
- Mobile-first approach for all components and layouts.
- All animations must use **Framer Motion** — no standard CSS animations (no `transition`, `@keyframes`, etc.).
- Aim for a highly luxurious, premium feel in every interaction.

## Architecture
- All UI components go in `src/components/`.
- Keep components modular — one responsibility per component.
